\addcontentsline{toc}{section}{27. Локальные экстремумы}

\vspace{3ex}

\hspace{-3.5ex}\underline{\textbf{Определение:}} $x_0$ \emph{точка локального экстремума} функции $f$, если $\exists \delta: \forall x, \; 0<|x-x_0|<\delta$ выполнено $f(x) < f(x_0)$, либо $f(x) > f(x_0)$.
\index{Локальный экстремум}

\vspace{3ex}

\hspace{-3.5ex}\underline{\textbf{Теорема}} \emph{(необходимое условие экстремума)}:

Пусть $x_0$ - точка локального экстремума функции $f$ и $\exists f^{\prime}(x_0)$ тогда $f'(x_0)=0$.

\vspace{3ex}

\hspace{-3.5ex}\underline{\textbf{Теорема}} \emph{(достаточное условие экстремума)}:

Если $\exists \delta>0,$ что $f$ непрерывна в $\mathbf{B}_{\delta}(x_0), \; \exists f^{\prime}(x)$ при $0 < |x-x_0| < \delta,$ \, а в самой точке $x_0$ производная $f^{\prime}(x_0)$ равна нулю, либо не существует. Тогда, если $f^{\prime}(x_1)\cdot f^{\prime}(x_2) < 0$ для $\forall x_1, x_2: \, x_0 - \delta < x_1 < x_0 < x_2 < x_0 + \delta,$ то $x_0$ - точка локального экстремума.

\vspace{3ex}

\hspace{-3.5ex}\underline{\textbf{Теорема:}} Пусть $\exists f^{\prime}(x)$ при $|x-x_0|<\delta, \,$ $f^{\prime}(x_0)=0, \, f^{\prime \prime}(x_0) < 0 \, (>0) \, \Rightarrow \, x_0$ - локальный максимум (минимум).

\vspace{3ex}

\hspace{-3.5ex}\underline{\textbf{Теорема:}} Пусть $\exists \,\,\, f^{\prime}(x), f^{\prime \prime}(x), \ldots ,f^{(n - 1)}(x)$ при $|x - x_0| < \delta,$ и выполнено:

$f^{\prime}(x_0) = f^{\prime \prime}(x_0) = \, \ldots  \, = f^{(n-1)}(x_0) = 0, \, f^{(n)}(x_0) \neq 0. $ Тогда, если
\vspace{2ex}

$
\begin{cases}
n = 2k, \, \text{то} \, x_0 \,\, \text{является точкой локального экстремума,}\\
n = 2k+1, \, \text{то} \,  x_0 \,\, \text{не является точкой локального экстремума \emph{(точка перегиба)}.}
\end{cases}\!\!\!\!\!\!
$
