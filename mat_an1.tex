\hspace{-3.5ex}\underline{\textbf{Утверждение}} \emph{(Метод математической индукции):}
\vspace{1ex}

\index{Метод!математической индукции|textbf}
 Пусть $M \subset \mathds{N}$ - такое множество, что:
\begin{enumerate}
\item [1)] $1 \in M$ \emph{(база индукции)};
\item [2)] $\forall n \in \mathds{N}$ из того, что $n \in M$ следует, что $(n+1) \in M$ \emph{(индукционный переход)};
\vspace{2ex}

Тогда $M=\mathds{N}$.
\end{enumerate}

\task \textit{(неравенства между "обыкновенными средними")}
\vspace*{2ex}

Пусть $a_i \in \mathds{R}, \, a_i > 0, \, \forall i =1,\ldots,n$. Обозначим
\vspace*{0.5ex}

$$A_n = \dfrac{a_1 + a_2 + \ldots + a_n}{n}, \qquad G_n = \sqrt[n]{a_1\cdot a_2 \cdot \ldots \cdot a_n}, \qquad \Gamma_n = \dfrac{n}{\frac{1}{a_1} + \frac{1}{a_2}+\ldots + \frac{1}{a_n}}.$$
\vspace{-1ex}

Докажите, что $A_n \, \geqslant \, G_n \, \geqslant \, \Gamma_n \;\;\; \text{для} \;\; \forall n \in \mathds{N}, \, n \ge 2.$

\vspace{3ex}

\underline{\textbf{Замечание:}} Выражения $A_n$ называется \emph{средним арифметическим}, $G_n$ - \emph{средним геометрическим}, $\Gamma_n$ - \emph{средним гармоническим} чисел $a_1, a_2, \ldots ,a_n.$

\vspace{2ex}

\underline{\textbf{Замечание:}} Отметим, что равенство в данных неравенствах возможно тогда и только тогда, когда выполнено: $a_1 = a_2 = \ldots = a_n.$

\begin{flushleft}\textbf{\Large{Метод математической индукции}}\end{flushleft}
\index{Метод!математической индукции}

\emph{Утверждение справедливо для всякого натурального $n$, если: 1) оно справедливо для $n=1$ и 2) из справедливости утверждения для какого-либо произвольного натурального $n=k$ следует его справедливость для $n=k+1$.}

\vspace{3ex}

$\blacktriangleright$
Предположим противное, т.е. предположим, что утверждение справедливо не для всякого натурального $\emph{n}$. Тогда существует такое натуральное $\emph{m}$, что:
\begin{enumerate}
\item утверждение для $\emph{n=m}$ несправедливо,
\item для всякого $\emph{n}$, меньшего $\emph{m}$, утверждение справедливо (иными словами, $\emph{m}$ есть первое натуральное число, для которого утверждение несправедливо).
\end{enumerate}

Очевидно, что $\emph{m > 1}$, так как для $\emph{n = 1}$ утверждение справедливо (условие 1). Следовательно, $\emph{m - 1}$ \textbf{---}натуральное число. Выходит, что для \textsf{\textsl{натурального}} числа $\emph{m - 1}$ утверждение справедливо, а для следующего натурального числа $\emph{m}$ оно несправедливо. Это противоречит условию 2).
$\blacksquare$

\vspace{5ex}

Конечно, при доказательстве принципа математической индукции мы пользовались тем, что \emph{в любой совокупности натуральных чисел содержится наименьшее число}. Легко видеть, что это свойство в свою очередь можно вывести как следствие из принципа математической индукции. Таким образом, оба эти предположения равносильны. Любое из них можно принять за одну из аксиом, определяющих натуральный ряд, \textbf{---} тогда другое будет теоремой. Обычно за аксиому принимают сам принцип математической индукции,называя его \emph{\textbf{аксиомой натуральных чисел}}.
