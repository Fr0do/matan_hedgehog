\vspace{1ex}

\hspace{-3.5ex}\underline{\textbf{Определение:}} Функция $f$ \emph{возрастает} в точке $x=x_0$, если
$\exists \delta(x_0) > 0$, что выполнено $f(x) < f(x_0)$ при $x_0 - \delta < x < x_0$, и $f(x) > f(x_0)$ при $x_0 < x < x_0 + \delta$.

\vspace{2ex}

\hspace{-3.5ex}\underline{\textbf{Определение:}} Функция $f$ \emph{убывает} в точке $x = x_o$, если
$\exists \delta(x_0) > 0$, что выполнено $f(x)>f(x_0)$ при $x_0 - \delta < x < x_0$, и $f(x) < f(x_0)$ при $x_0 < x < x_0 + \delta$.

\vspace{2ex}

\hspace{-3.5ex}\underline{\textbf{Утверждение:}} Пусть функция $f$ дифференцируема на интервале $(a; b)$.
Между характером монотонности данной функции и знаком её производной на этом интервале имеется следующая взаимосвязь:
$$f^{\prime}(x) > 0 \, \emph{(\text{дост})} \; \Rightarrow \, f(x)\!  \uparrow \; \emph{(\text{необх})} \; \Rightarrow \,  f^{\prime}(x) \geqslant 0; \quad f^{\prime}(x) \geqslant 0 \, \Rightarrow \, f(x)\!\!\nearrow \; \Rightarrow \,  f'(x) \geqslant 0;$$

$$f^{\prime}(x) = 0 \, \Rightarrow \, f(x) = \textbf{const} \, \Rightarrow \, f^{\prime}(x) = 0;$$

$$f^{\prime}(x) < 0 \, \Rightarrow \, f(x)\! \downarrow \; \Rightarrow \, f^{\prime}(x) \leqslant 0; \quad f^{\prime}(x) \leqslant 0 \, \Rightarrow \, f(x)\!\! \searrow \; \Rightarrow \, f^{\prime}(x) \leqslant 0;$$

\vspace{2ex}

\hspace{-3.5ex}\underline{\textbf{Замечание:}} Отметим, что строгая положительность производной не является необходимым условием возрастания функции.

\textit{Например, функция $f(x) = x^{3}$ в точке $x=0$ возрастает, но $f^{\prime}(x) = 0$.}

\hspace{-3.5ex}\underline{\textbf{Определение:}} Функция $f$ \emph{выпукла вверх на множестве $\textbf{X}$}, если $\forall x_1, x_2 \in \{x\};$

\hspace{-3.5ex}$\forall \alpha_1, \alpha_2 \geqslant 0 \, : \, \alpha_1~+~\alpha_2~=~1 \, \Rightarrow$ \vspace{-1ex}$$ f(\alpha_1 x_1 + \alpha_2 x_2) \geqslant \alpha_1 f(x_1) + \alpha_2 f(x_2).$$
\vspace{-2ex}

\hspace{-3.5ex}\underline{\textbf{Определение:}} Функция $f$ \emph{выпукла вниз на множестве $\textbf{X}$}, если $\forall x_1, x_2 \in \textbf{X};$

\hspace{-3.5ex}$\forall \alpha_1, \alpha_2 \geqslant 0 \, : \, \alpha_1~+~\alpha_2~=~1 \Rightarrow$ \vspace{1ex}
$$\hspace{-46ex} f(\alpha_1 x_1+\alpha_2 x_2) \leqslant \alpha_1 f(x_1)+\alpha_2 f(x_2).$$
\vspace{-3ex}

\hspace{-3.5ex}\underline{\textbf{Теорема}} \emph{(критерий выпуклости):}

Пусть функция $f$ дважды дифференцируема на множестве $\textbf{X}$. Тогда, для того, чтобы данная функция была выпуклой вверх (выпуклой вниз), необходимо и достаточно, чтобы $f^{\prime \prime}(x) \le 0 \,$ $\bigl(f^{\prime \prime}(x) \ge 0 \,\bigr)$ для всех $x$ из множества $\textbf{X}$. Если же $f^{\prime \prime}(x) < 0 \,$ $\bigl(f^{\prime \prime}(x) > 0 \,\bigr),$ то этого достаточно, чтобы гарантировать \textsf{\textsl{строгую}} выпуклость вверх (вниз) данной функции на этом множестве.

\hspace{-3.5ex}\underline{\textbf{Определение:}} Точка $x$ является \emph{точкой перегиба функции} \emph{$f$}, если в данной точке $f$ меняет направление выпуклости.
\index{Точка перегиба}
\vspace{2ex}

\hspace{-3.5ex}\underline{\textbf{Теорема:}} Пусть функция $f$ дважды дифференцируема на множестве
$\{0<|x-x_0|<\delta\},\,$ $f^{\prime \prime}(x_0)~=~0$ или $\nexists f^{\prime \prime}(x_0),$ и, кроме того,
$f^{\prime \prime}(x_1) \cdot f^{\prime \prime}(x_2)<0$, где $x_0 - \delta < x_1 < x_0 < x_2 < x_0 + \delta.$ Тогда $x_0 ~-~\text{точка перегиба}$.
